# Arpan

Data engineer who likes building small, complete tools rather than leaving
things half-finished. Most of what's here started as "let me just check if
this works" and turned into something I kept using.

I also put together training material for a Python / GenAI fundamentals
program, teaching through real production code instead of toy examples.

## A few projects worth a look

**[market-email-project](https://github.com/arpan2711/market-email-project)**
Stock market tools built up piece by piece: a CLI ticker, a local
refresh-to-update price/news page, and a RAG chatbot that answers questions
against your own saved articles and portfolio notes.

**[chess-recall](https://github.com/arpan2711/chess-recall)**
A chess pattern-recognition trainer - study a position, rebuild it from
memory, get scored. Includes a data pipeline that builds the position bank
from real Lichess games.

**[habits-tracker](https://github.com/arpan2711/habits-tracker)**
A full-stack habit tracker with a journaling layer (Next.js, Supabase,
Postgres). Tap to mark a day done, hold to leave a note.

**[ai_training](https://github.com/arpan2711/ai_training)**
GenAI fundamentals training materials, including a RAG + tool-calling
support bot capstone project.

**[adv-python](https://github.com/arpan2711/adv-python)**
Advanced Python concepts taught through real snippets from Flask, Scrapy,
LangChain, and pydoit rather than isolated examples.
